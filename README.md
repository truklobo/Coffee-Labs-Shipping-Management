# Coffee-Labs-Shipping-Management
A full-stack web app for managing shipmends and visualizing logistics data. Built wirth Python Flask, Bootstrap, and FusionCharts.js.

Features:
- Create, view, edit, and delete shipment records through clean, user-friendly forms
- Advanced search interface with dynamic result tables and filter controls for quick lookups
- Interactive charts and analytics dashboards for shipment trends and performance metrics
- Bootstrap-based layout with custom CSS/JS for desktop and mobile
- Jinja2 base layout with reusable includes for consistent UI across pages

Stack:
- backend: Python, Flask
- frontend: Bootstrap, JavaScript, FusionCharts.js
- templating: Jinja2
- db: MySQL
- deployment: Gunicorn, Docker
